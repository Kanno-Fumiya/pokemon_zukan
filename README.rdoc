= pokemon_zukan

ポケモン図鑑のgemです。
ポケモンXYの全国図鑑に対応しています。

== Install

 gem install pokemon_zukan

== Use

 require 'pokemon_zukan'

 pokemon = PokemonZukan.find("001")
